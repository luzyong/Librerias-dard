# Librerias-dard
Código que se puede reutilizar en la aplicación final
main.dart es un archivo que solo prueba los tipos de datos en dart y cómo funcionan. Se remonta a marzo de 2021, cuando recién comenzaba a aprender el lenguaje.
persona.dart es una respuesta aun ejercicio de POO que utilizaba los constructores y el encapsulamiento para poder calcular el IMC de una persona.
expresionregular.dart es un archivo que prueba cómo implementar expresiones regulares en dart.
